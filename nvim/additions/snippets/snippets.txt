* Creating a new snippet: 
	- Create a new file in ~/vim/additions/snippets/ ( "snip.filetype" ).
	- Type the snippet code into the file.
	- In .vimrc under SNIPPETS remap the snippet to the desired key combination following the example:

		```nnoremap yourKeys :-1read ~/.vim/additions/snippets/snippetFileName.filetype<CR>```
		
	- You can add keys to navigate the snippet at the end of the line as shown:
 
		```nnoremap yourKeys :-1read ~/.vim/additions/snippets/snippetFileName.filetype<CR>j2w```
